During class today we added some extensions to our Visual Studio Code program.

  • Auto Rename Tag
  • indent-rainbow
  • Prettier-Code formatter
  • Live Server (some of us had this one already)
This was super helpful to make our VS Code looked like Julie's so we could follow along easier. Some of the functionality that these add made me so happy.
